AI Smart Watering System

This project designs an IoT-based plant watering system that collects real-time soil moisture using ESP32 sensors, uploads data to a Flask server, stores irrigation history, and sends structured JSON to GPT for intelligent watering analysis. The system allows manual control & configuration through a web dashboard and supports future integration with AI-based watering recommendations.

Motivation

Motivation

Indoor plants are easy to love but hard to maintain. Without proper knowledge, many people water plants either too frequently or not enough, leading to wilted or overwatered roots. Traditional plant care relies heavily on intuition rather than data, resulting in inconsistent growth and unnecessary water waste.

Lack of watering ‚Üí Plants wilt

Over-watering ‚Üí Root rot & yellow leaves

Our goal is to introduce a data-driven and AI-assisted smart watering system that continuously monitors soil moisture using IoT sensors, logs watering history, and leverages GPT to provide intelligent irrigation recommendations. This approach transitions plant care from experience-based decisions to scientific, personalized, and scalable plant management ‚Äî a concept highly applicable to future smart agriculture.

System

Our smart plant watering system combines IoT sensors, AI vision analysis, and an interactive web interface to provide intelligent plant care recommendations. The system continuously monitors environmental conditions and makes data-driven watering suggestions through a Large Language Model, delivering alerts and guidance via a user-friendly website.

Results

Our system integrates data from multiple IoT sensors, weather forecasting APIs, and AI plant analysis to generate accurate watering recommendations. Below are the key results illustrating each component of the pipeline.

Sensor Output ‚Äî Real-Time Data Acquisition

The ESP32-based soil moisture and humidity sensors provide continuous real-time readings. These values allow the LLM to understand the current condition of the plant's environment and determine whether watering is required. The example above shows live sensor readings visualized on the dashboard.

Web Interface ‚Äî User Dashboard

The web interface provides a clean and intuitive view of the system state. Users can monitor plant status, weather forecasts, soil moisture, and air humidity. The dashboard updates automatically as new data is received from the IoT device.

AI Watering Recommendation

Using plant images, weather forecasts, soil moisture, and humidity levels, the Large Language Model analyzes whether watering is necessary and determines the recommended amount. The example above shows an LLM-generated decision based on 24-hour maximum temperature, rainfall prediction, and detected soil dryness.

Historical Data ‚Äî Trend Tracking

The system stores historical measurements of soil moisture and watering events. Users can visualize long-term trends, helping them understand plant behavior under different conditions. This enables more scientific plant care and supports future model improvements.
